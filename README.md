# 🎓 Öğrenci Bilgi Sistemi (C++)

Bu proje, bir sınıftaki öğrencilerin not verilerini ve devamsızlık durumlarını dosyalardan okuyarak harf notu, ortalama ve geçme/kalma durumunu hesaplayan bir C++ programıdır.

## 🛠️ Nasıl Çalıştırılır?

Bu projeyi kendi bilgisayarınızda çalıştırmak için aşağıdaki adımları takip edebilirsiniz:

### 1. Gereksinimler
* Bilgisayarınızda bir C++ derleyicisinin (GCC/MinGW, Clang, MSVC vb.) veya bir IDE'nin (Code::Blocks, VS Code, Dev-C++) kurulu olması gerekir.

### 2. Projeyi İndirme
Projeyi bilgisayarınıza indirmek için sağ üstteki yeşil **Code** butonuna tıklayıp **Download ZIP** diyebilir veya Git kullanıyorsanız terminale şu komutu yazabilirsiniz:
```bash
git clone [https://github.com/KULLANICI_ADINIZ/ogrenci-bilgi-sistemi.git](https://github.com/KULLANICI_ADINIZ/ogrenci-bilgi-sistemi.git)
